# Sentimental-analysis-for-RNN-

This project focuses on sentiment analysis of movie reviews using a Recurrent Neural Network (RNN). The IMDB dataset is used to train a deep learning model capable of understanding and classifying textual data.

Key features:

Data preprocessing and text cleaning
Tokenization and sequence padding
RNN-based deep learning model
Model training and evaluation
Performance analysis on test data

Technologies used:

Python
TensorFlow / Keras
NumPy, Pandas
Natural Language Processing (NLP)
